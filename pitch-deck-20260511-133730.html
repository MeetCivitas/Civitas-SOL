The Solution · 04

Four cooperating privacy layers.
One Anchor program.

Each layer addresses a distinct privacy threat. The system breaks only if every layer is compromised simultaneously.

LAYER 01
Nillion nilDB
3 of 3 secret shared cluster. Salary columns stored as %allot. No single operator sees the table.
LAYER 02
Nillion nilCC TEE
AMD SEV-SNP enclave. Pinned launch measurement. Payroll Poseidon commitments computed inside the TEE.
LAYER 03
Groth16 ZK gate
256 byte proofs verified on chain through alt_bn128_pairing syscalls. About 200K CU per redemption.
LAYER 04
MagicBlock PER
TEE validated Private Ephemeral Rollup. USDC settles in sub 50ms. On chain gate never linkable to payout.
How It Works · 05

End to end flow.
Five steps, twelve seconds.

01
Employer drops payroll spec.
Encrypted into nilDB cluster as %allot columns.
02
nilCC TEE computes.
Poseidon commitments and Merkle root inside SEV-SNP enclave.
03
Anchor program publishes root.
Only the Merkle root and commitments hit the chain. No amounts.
04
Employee proves redemption.
256 byte Groth16 proof, verified natively via alt_bn128_pairing.
05
MagicBlock PER settles.
Sub 50ms private USDC transfer. Gate unlinkable to payout.
No salary on chain.No master credential leaves the browser.No single operator sees the full table.
